# Default Theme

The theme object is where you define your application's color palette, type scale, font stacks, breakpoints, border radius values, and more.

This library uses [Stitches](https://stitches.dev/docs/introduction) and [Radix Colors](https://www.radix-ui.com/docs/colors/palette-composition/the-scales).

[<Button size='medium'>View Source</Button>](https://github.com/smurph7/stitch-ui/blob/main/src/theme/stitches.config.ts)

### Dark Theme

Light scales apply their CSS variables to the `:root` element. Dark scales apply their CSS variables to the `.dark-theme` class.

### Changing Theme

Use hooks to toggle the theme at the click of a button!

```javascript
import { darkTheme } from '../stitches.config';

const [theme, setTheme] = React.useState('theme-default');

React.useEffect(() => {
  document.body.classList.remove('theme-default', darkTheme);
  theme === 'theme-default' ? theme : theme.className;
}, [theme]);

return (
  <Button
    onClick={() =>
      setTheme(theme === 'theme-default' ? darkTheme : 'theme-default')
    }
  >
    Toggle theme
  </Button>
);
```

### Breakpoints

A predefined set of commonly used breakpoints for responsive styling. View [Stitches](https://stitches.dev/docs/responsive-styles) for usage tips.

```javascript
media: {
    bp1: '(min-width: 0px)',
    bp2: '(min-width: 568px)',
    bp3: '(min-width: 768px)',
    bp4: '(min-width: 1024px)',
    bp5: '(min-width: 1280px)',
    bp6: '(min-width: 1536px)'
  },
```

### Spacing

The `space` key allows you to customize the global spacing and sizing scale. By default these spacing values can be referenced by the `padding`, `margin`, and `top`, `left`, `right`, `bottom` styles.

```javascript
space: {
    1: '4px',
    2: '8px',
    3: '12px',
    4: '16px',
    5: '24px',
    6: '32px',
    7: '48px',
    8: '64px',
    9: '96px',
    10: '128px',
    11: '160px'
  }
```

By default, Stitch-UI includes a comprehensive numeric spacing scale inspired by Tailwind CSS. The values are proportional, so 1 spacing unit is equal to 4px by default in common browsers.

### Sizes

The `sizes` key allows you to customize the global sizing of components. By default these sizes value can be referenced by the `width`, `height`, and `maxWidth`, `minWidth`, `maxHeight`, `minHeight` styles.

```javascript
sizes: {
    1: '4px',
    2: '8px',
    3: '12px',
    4: '16px',
    5: '24px',
    6: '32px',
    7: '48px',
    8: '64px',
    9: '96px',
    10: '128px',
    11: '160px'
  }
```

### Border radius

Smooth corner radius values.

```javascript
 radii: {
    1: '4px',
    2: '6px',
    3: '8px',
    4: '12px',
    round: '50%',
    pill: '9999px'
  }
```
